# Mirror
It is a website that is going to be made into an application to check your mental, physical and medical health.

Mental
--------
- tracks happiness level with user input
- tracks stress level with user input
- provides journal to write thoughts and/or feelings
- provides direct contact to therapist via text
- shows progress of happiness level

Physical
---------
- tracks weight with user input
- tracks calorie and macromolecule intake with user input
- provides bmi and bf percentage

Medical
---------
- provides direct contact with physician
- keeps medical information
- can type in symptoms and finds out if you have any illness with user input
